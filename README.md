# 커뮤니티 프로젝트 - 프론트엔드 (Frontend)

이 프로젝트는 커뮤니티 서비스의 프론트엔드 부분으로, 사용자가 게시글을 작성하고 댓글을 달며 소통할 수 있는 웹 애플리케이션입니다.
HTML, CSS, Vanilla JavaScript를 사용하여 구현되었습니다.

## 🛠 기술 스택

- **언어**: HTML5, CSS3, JavaScript (ES6+)
- **스타일링**: Vanilla CSS
- **통신**: Fetch API (백엔드 API 연동)

## ✨ 주요 기능

- **회원 관리**
  - 회원가입 및 로그인
  - 프로필 수정 (프로필 이미지, 닉네임)
  - 비밀번호 변경
  - 회원 탈퇴 (작성한 게시글 및 댓글 자동 삭제 처리)

- **게시판 기능**
  - 게시글 목록 조회 (조회수 표시)
  - 게시글 상세 보기
  - 게시글 작성 (이미지 업로드 지원)
  - 게시글 수정 및 삭제

- **소통 기능**
  - 댓글 작성 및 삭제
  - 게시글 좋아요 기능 (예정)

## 📂 폴더 구조

```
2-junwoo-community-fe/
├── assets/         # CSS 스타일 및 정적 리소스
├── components/     # 헤더 등 재사용 가능한 UI 컴포넌트
├── pages/          # 페이지별 JavaScript 로직
├── *.html          # 각 페이지 HTML 파일
└── README.md       # 프로젝트 설명
```

## 🚀 실행 방법

1. 이 디렉토리를 로컬 서버(Live Server 등)로 실행하거나 브라우저에서 직접 HTML 파일을 엽니다.
2. 백엔드 서버가 실행 중이어야 정상적으로 기능이 작동합니다.
3. `index.html` 또는 `login.html`로 접속하여 서비스를 이용합니다.
